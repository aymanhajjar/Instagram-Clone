# Insta Now

An Instagram clone built using Django and Javascript.

### Features

- Login/Sign up
- Post
- Like, comment and save
- Notifications
- Edit Profile
- Explore page
- Infinite scroll
- Follow/Unfollow

### How to Run

- Install requirements

```sh
   pip install -r requirements.txt
```

- Run

```sh
   python manage.py runserver
   python3 manage.py runserver # if using python 3
```
